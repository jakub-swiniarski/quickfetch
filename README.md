## How to install?
### Compile & install
```shell
git clone https://github.com/jakub-swiniarski/quickfetch.git
cd quickfetch
sudo make install
```

### How to run?
Use this command:
```shell
qf
```

### How to uninstall?
Go inside the cloned repository and run: <br/>
```shell
sudo make clean uninstall
```

## Configuration
You can configure QuickFetch by editing the source code.
